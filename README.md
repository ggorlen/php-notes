# PHP Notes

Simple notes app in vanilla PHP with SQLite3.


### Setup

```
$ sudo apt install php8.1-sqlite3
$ php -v
PHP 8.1.2-1ubuntu2.11 ...
```


### Development

```
php -S localhost:8000
```
